Программа представляет собой консольный телефонный справочник, позволяющий работать с контактами, сохранёнными в файле формата JSON.

После запуска приложения пользователю отображается меню с доступными действиями:

1. Открыть файл
2. Сохранить файл
3. Показать все контакты
4. Создать контакт
5. Найти контакт
6. Изменить контакт
7. Удалить контакт
8. Выход

Работа программы строится следующим образом:

1. Открытие файла
Пользователь вводит имя JSON-файла, который должен называться: contacts.json (приложен в репозитории)
Если файл существует и имеет корректный формат (список контактов), данные загружаются в память.
Если файл не найден или содержит некорректный JSON, выводится сообщение об ошибке.

2. Сохранение файла
Все изменения записываются обратно в JSON-файл.
Если файл не был открыт, сохранение невозможно.

3. Просмотр контактов
После открытия файла пользователь может вывести список всех контактов.
Каждый контакт отображается в формате:
ID) Имя – Номер телефона; Комментарий

4. Добавление контакта
Пользователь вводит имя, номер телефона и комментарий.
Номер телефона проходит проверку на корректность (должен быть числом).
Контакт добавляется в список и получает новый ID.

5. Поиск контакта
Пользователь вводит текст запроса.
Поиск выполняется по всем полям контакта (имя, номер, комментарий).
Если совпадения найдены — они выводятся на экран.
Если совпадений нет — выводится соответствующее сообщение.

6. Изменение контакта
Пользователь вводит ID контакта, который хочет изменить.
Далее можно изменить имя, номер или комментарий.
Любое поле можно оставить без изменений, нажав Enter.

7. Удаление контакта
Пользователь вводит ID контакта.
Если контакт найден, он удаляется из списка.

8. Выход из программы
При выходе, если есть несохранённые изменения, программа предлагает сохранить их.
После этого приложение завершает работу.

Программа работает в цикле до тех пор, пока пользователь не выберет пункт «Выход».
